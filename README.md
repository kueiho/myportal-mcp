# my-portal-mcp

MCP (Model Context Protocol) Server，提供 AI 助手存取 my-portal 資料的工具介面。

## 安裝

```bash
pnpm install
```

## 開發

```bash
pnpm dev
```

## 編譯與啟動

```bash
# 編譯 TypeScript
pnpm build

# 啟動編譯後的服務
pnpm start
```

## 指令總覽

| 指令 | 說明 |
|------|------|
| `pnpm dev` | 開發模式（tsx watch） |
| `pnpm build` | 編譯 TypeScript |
| `pnpm start` | 啟動編譯後的服務 |
